1. **Usando o operador de módulo (%):**

```python
def is_even(number):
    return number % 2 == 0
```

O operador de módulo retorna o resto da divisão de um número por outro. Se o resto for 0, o número é par.

2. **Verificando se o último dígito é par:**

```python
def is_even(number):
    last_digit = number % 10
    return last_digit % 2 == 0
```

Um número é par se seu último dígito for 0, 2, 4, 6 ou 8.

3. **Bitwise AND (&):**

```python
def is_even(number):
    return (number & 1) == 0
```

O operador bitwise AND retorna o último bit do número. Se o último bit for 0, o número é par.
